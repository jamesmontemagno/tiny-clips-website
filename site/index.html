Install with Homebrew (full directions)

Perfect for terminal-first workflows and reproducible setup.

  1. Install Homebrew (if you don't already have it): /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Finish Homebrew shell setup: eval "$(/opt/homebrew/bin/brew shellenv)" On Intel Macs, use eval "$(/usr/local/bin/brew shellenv)".
  3. Install Tiny Clips: brew install --cask tiny-clips
  4. Verify installation: brew list --cask tiny-clips
  5. Update later: brew update && brew upgrade --cask tiny-clips
  6. Uninstall if needed: brew uninstall --cask tiny-clips

Configurable for your workflow

Tiny Clips includes practical settings for creators and developers: choose save location, copy captures to clipboard, reveal files in Finder, tweak formats, quality, frame rates, countdowns, and trimmer behavior.

  • Save destination: Choose any folder (Desktop by default).
  • Clipboard flow: Copy captures automatically after save.
  • Finder handoff: Reveal files immediately when export finishes.
  • Screenshot controls: Format, quality, and scaling options.
  • Video controls: 24/30/60 fps plus system and microphone audio toggles.
  • GIF controls: Tune frame rate and max width for lightweight sharing.
  • Editing flow: Open screenshot editor or trimmer after capture.
  • Countdowns: Optional timer before video and GIF recording starts.
